Evaluate models and get rewards

A model evaluation platform based on blockchain

When a machine learning model is trained, it needs evaluations with a large number of testing data to validate its effectiveness. However, model developers usually have limited testing data for such evaluations. Furthermore, users may lack of trust in a machine learning model because of validations with limited data from the model developer as well as other reasons. This platform aims to solve such problems. We create a marketplace for both testing data and machine learning models. On this platform, a machine learning model can be fully evaluated with numerous testing data from all over the world, while data owners can earn rewards from such evaluations. Owners of machine learning models can also sell their fully evaluated models to potential users on this platform. All evaluation results are stored in a blockchain, which guarantees that the results cannot be tampered by any individual. The evaluation with numerously diverse testing data and blockchain technology boost trustworthy AI. {% comment %} Two issues: trust and who controls Machine learning technology is being developed and used at an ever increasing rate. However, without access to quality testing data, potential buyers of the technology have no way to assess its performance without trusting the original engineer. This is the problem which this platform is to solve. which can be used to verify the quality of machine learning models. However, one problem remains: who controls the platform? If we have to trust a third party to provide us with the correct metrics then we are in no better a situation than before. Thus, we store the results on a blockchain. This way, the results cannot be tampered with by any single individual, and we can be certain that the presented metrics are true, even in a trustless setting. {% endcomment %}

Trustworthy AI
 In an increasingly dynamic society, where technology has become a central part of human interaction, the importance of creating a trustworthy source of artificial intelligence (AI) has become increasingly apparent. This platform revolves around the notion of blockchains and how they can be used to explain AI and its’ output to ultimately increasing user trust.

 Blockchain technology refers a to a shared technological system that allows any participants within a business network to see the system of record (also known as ledger) and permits transactions gathered into blocks (of the blockchain) and recorded. The blockchain process provides a transparent platform by introducing the notion of an autonomous self-improvement system that essentially helps users solicit machine learning models for a given fee, as outlined in the terms and conditions in a smart contract. Further, blockchains also allow the resulting ledger to be securely accessed by different servers. This is where the notion of blockchain boosting trustworthy, as the title of our project suggests, where trust between individuals is not necessary as the functionalities of using blockchain technology to evaluate and test Machine Learning models use cryptographic verification for its’ transactions between users who upload models and other users who upload datasets to test those models.

 Machine learning (ML) is widely used to extract knowledge from data for the data-driven decision making. In essence, there are two main processes involved in developing a trained ML model using blockchains (Kurtulmus, A. Besir, Daniel, Kenny (27 Feb 2018)). Firstly, training the model using datasets adjusts the model weights to increase the accuracy of the model. The second process involved to produce a highly trained ML model is testing whereby independent data sets are uploaded by users to the system, which is then used for testing different problems related to the trained model. These different problems are dependent upon the type of problem that is being solved at hand. Certain metrics such as accuracy, recall, precision, and sensitivity to many other metrics can be used to evaluate the success of a given model. Together, these processes help validate the model thereby boosting trust between users.
